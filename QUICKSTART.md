# 2026 최신동향 1분 실행법

설치형 프로그램처럼 쓰는 로컬 웹앱입니다. 압축을 풀고 시작 파일만 더블클릭하세요.

## Mac

1. `start.command` 더블클릭
2. 보안 경고가 뜨면 `start.command`를 우클릭 또는 Control+클릭 → `열기`
3. 브라우저가 열리면 `카메라 권한`을 허용
4. `보호 시작`을 누르고 `비상 전환 테스트` 또는 `데모 시나리오 시작`으로 확인

## Windows

1. `start.bat` 더블클릭
2. 브라우저가 열리면 `카메라 권한`을 허용
3. `보호 시작`을 누르고 `비상 전환 테스트` 또는 `데모 시나리오 시작`으로 확인

## Python이 없다고 나오면

`https://www.python.org/downloads/`에서 Python을 설치한 뒤 다시 시작 파일을 더블클릭하세요.

## 안전 범위

- 카메라 화면은 내 컴퓨터 안에서만 처리됩니다.
- 서버 업로드 없음.
- 얼굴 인식 없음.
- 계정/로그인 없음.

## 종료 방법

브라우저 탭을 닫고, 같이 열린 터미널/명령 프롬프트 창에서 `Ctrl + C`를 누르거나 창을 닫으면 종료됩니다.
